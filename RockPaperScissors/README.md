# Rock-Paper-Scissors: Last Life

A high-stakes Rock-Paper-Scissors game. Face off against the computer with only 3 lives — lose a round, lose a life. Survive as long as you can.

**Engine:** Godot 4.x | **Genre:** Casual / strategy

## How to Play

- Pick rock, paper, or scissors each round
- Standard rules: Rock beats Scissors, Scissors beats Paper, Paper beats Rock
- Winning or drawing keeps you going — losing costs a life
- Game over when all 3 lives are gone
