# Wizard Dungeon Escape

A top-down 2D dungeon puzzle/action game. You play as a wizard trapped in a dungeon, navigating rooms filled with enemies and walls. Use your limited set of magical spells — Line Blast, Smite, and Phase Walk — to survive and reach the exit door.

**Engine:** Godot 4.x | **Genre:** Dungeon puzzle / action

## How to Play

- WASD / Arrow keys to move (8-directional)
- **Spell 1 — Line Blast** (`1` / LMB): Fire a beam that kills all enemies in a line
- **Spell 2 — Smite** (`2` / RMB): Instantly kill the nearest enemy
- **Spell 3 — Phase Walk** (`3` / Space): Walk through walls for 5 seconds
- Each spell has 5 charges per level
- Reach the exit door to win
